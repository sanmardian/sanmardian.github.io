Asset Management Project

  • Tech Stack: PHP, HTML, CSS, Javascript, Framework Codeigniter, Bootstrap
  • URL: Project Link

This project involves the development of a landing page utilizing the headless CMS Contentful and the Codeigniter framework. Contentful serves as the backend for content management, providing a user-friendly interface for creating, editing, and publishing content. The headless approach allows for flexibility in delivering content to various platforms and devices. Codeigniter framework is used to create a responsive and visually appealing design, with pre-built components and styling options that streamline the development process. The combination of Contentful and Codeigniter ensures an efficient workflow, enabling the team to focus on content creation and design without worrying about the technical implementation. The result is a professional and engaging landing page that effectively showcases the desired content and captures the attention of the target audience.
